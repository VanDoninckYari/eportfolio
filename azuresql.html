Azure SQL Database Security & Back-up

Voor deze opdracht heb ik op Azure een SQL Database opgezet die gelinkt werd aan een Generic Crud App. Deze crud app toonde data over titels van computerspellen die opgeslagen waren in de SQL database.

Een van de requirements van de opdracht was om security best practices toe te passen zoals toegang beperken tot de database en rollen voorzien voor de gebruikers. Een andere requirement was een back-up strategie voorzien indien er data of zelfs de hele database verwijderd zou worden. Dit werd dan getest met de crud app waar er knoppen waren voorzien om bepaalde records uit de database te verwijderen of aan te passen.

Dit was een hele interessante oefening voor het opzetten van security en back-up toepassingen op Azure als het managen van een SQL database en zijn data.
